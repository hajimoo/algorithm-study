# Time Complexity Notes

| Complexity | Name | 느낌 | 예시 |
|---|---|---|---|
| O(1) | 상수 | 입력 커져도 일정 | 단순 계산 |
| O(log N) | 로그 | 천천히 증가 | 이진탐색 |
| O(N) | 선형 | 한 번 순회 | for 한 번 |
| O(N log N) | 선형로그 | 정렬급 | sort |
| O(N^2) | 제곱 | 급격히 느려짐 | 이중 for |
| O(2^N) | 지수 | 폭발 | 완전탐색 |

## 1초 감각표
- N ≤ 500 → O(N^3) 가능
- N ≤ 2,000 → O(N^2) 가능
- N ≤ 100,000 → O(N log N) 가능
- N ≤ 10,000,000 → O(N) 필요
