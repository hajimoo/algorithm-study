# Python Libraries for Coding Tests

## 📦 Built-in
- 기본 문법 / 내장 함수 사용

---

## 🔁 itertools
- 반복되는 형태 데이터 처리
- 순열 / 조합 제공
- 완전탐색에서 코드가 간결해짐

---

## 🗂 heapq
- 힙(우선순위 큐) 구현
- 다익스트라 같은 최단경로 문제에서 사용

---

## 🔎 bisect
- 이진 탐색 관련 기능 제공
- 정렬된 배열에서 빠른 탐색 가능

---

## 📚 collections
- `deque` → 큐 구현
- `Counter` → 빈도수 계산

- 최대공약수 (`gcd`)
- 삼각함수
